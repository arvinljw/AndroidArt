# AndroidArt

本项目包括两部分

* [个人博客](https://arvinljw.github.io)中的demo
* 开发框架的介绍我这边已经换了个项目去整理了，与之前的也有些调整，项目地址：[AFBaseLibrary](https://github.com/arvinljw/AFBaseLibrary)

## 个人博客：

* Activity的理解；
* Service、Broadcast、ContentProvider的使用；
* Intent的理解；
* View的绘制和事件体系；
